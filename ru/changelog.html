Версия 0.9

Последняя версия Reunion 7

Детали

  • После долгого ожидания темы Basic и Classic наконец доступны на странице персонализации (в Панели управления). Благодарим Tech Stuff за решение этой проблемы и Xdek за помощь в реализации!
  • Точный установщик Windows 7 был портирован из Classic 7, благодаря Patrick и Xdek.
  • Всплывающие уведомления Windows 10 отключены (так как они неточны), хотя их можно снова включить.
  • Установлен улучшенный (модифицированный) Aerexplorer, благодаря Tech Stuff.
  • Улучшена грамматика во всех файлах readme и инструкциях RTF.
  • Удален обычный AWM и заменен на мод Windhawk (спасибо Tech Stuff).
  • Переход на DWM Tweaker (спасибо Tech Stuff).
  • Частично добавлена поддержка точек восстановления.
  • Реализованы ВСЕ общедоступные восстановления оболочки CPL Windows 7.
  • Частично восстановлен гаджет Windows Media Center.
  • Частично восстановлен официальный гаджет Погоды (только для демонстрационных целей!).
  • Исправлена небольшая неточность (где отсутствовал тег "Системная папка"). Спасибо Xdek.
  • Исправлена неточность, при которой вкладка "Избранное" по умолчанию была свернута.
  • Исправлена неточность, при которой отсутствовала папка "Недавние места".
  • Исправлена неточность, при которой присутствовала неработающая функция закрепления в "Избранном"; этой кнопки не было в W7 изначально.
  • Исправлена небольшая неточность на странице библиотек: было "Выберите элемент для просмотра сведений", теперь "... элементы".
  • Исправлена неточность, при которой PowerShell ISE был от W10, теперь он должен быть от W7.
  • Исправлена ошибка, при которой 12-часовой формат времени имел 0 перед часами 1-9, т.е. что-то вроде 09:30 исправлено на просто 9:30.
  • Наконец-то в меню "Пуск" появляются правильные, точные программы по умолчанию, как в W7 (например, Косынка, Windows Media Center).
  • Исправлена ошибка, при которой значок акцентного цвета "Морозный" отображался как значок "Автоматический" цвет.
  • Исправлена ошибка, при которой отсутствовали все схемы управления питанием (кроме "Сбалансированной"). Хотя это неточность, как напоминание об официальных днях, опция "Максимальная производительность" возвращена.
  • Исправлена серьезная проблема, при которой у учетных записей без прав администратора были сломанные элементы проводника (и другое).
  • Восстановлена программа "Сетевой проектор" W7 (только для демонстрационных целей), спасибо Petya и djamaliyev64 (за отзыв)!
  • Восстановлен экран ключа продукта Windows 7, спасибо ImSwordQueen и Brawllux!
  • Исправлена неточность, при которой индикатор загрузки устранения неполадок использовал стиль Metro вместо Aero, спасибо Petya!
  • Исправлена неточность рамок окон OpenGlass, которые неправильно сливались с остальным стеклом.
  • Исправлена проблема с грустным смайликом :( в меню "Пуск".
  • Отключено всплывающее окно "Поиск приложения в Магазине".
  • Извлечен оставшийся сетевой диск.
  • Реализованы лучшие благодарности в этом файле readme.
  • Исправлены проблемы, связанные с форматами времени.
  • Обновлен Winaero Tweaker.
  • Удален проблемный Windows Marketplace.
  • Реализован дополнительно оптимизированный OpenGlass (движок рендеринга стекла), который работает на 160% лучше. Спасибо Patrick и Tech Stuff.
  • Исправлена проблема, при которой Reunion7 был довольно медленным (даже после обновления драйверов). Отражения Aero Peek/Snap все еще присутствуют. ОГРОМНОЕ спасибо Patrick и Tech Stuff.
  • Исправлена ошибка, при которой "Недавние места" могли вызывать сбой explorer.exe.
  • "Недавние места" теперь работают благодаря Xdek!
  • Контроль учетных записей (UAC) теперь включен по умолчанию (как в W7).
  • Убрана путаница, которая была коллекцией batch-скриптов для реализации исправлений.
  • Гаджеты "Погода" и "Валюта" теперь работают. Спасибо, Patrick!
  • Удален неточный акцентный цвет "Автоматический".
  • Исправлены extremely minute неточности, которые подавляющее большинство людей не заметит. Спасибо, Xdek.
  • Исправлена неточность, при которой сочетание клавиш Win+Tab активировало меню W10.
  • Восстановлен точный ножницы W7.
  • Восстановлен точный проигрыватель Windows Media W7.
  • Исправлены некоторые неточности командной строки в установке ISO.
  • Множество ошибок и неточностей "Открыть с помощью" исправлены благодаря OpenWithEx от aubymori.
  • Полностью переписан этот файл readme.
  • Незначительные исправления ошибок.
